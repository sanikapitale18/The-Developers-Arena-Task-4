# Week 4 - Data Visualization Project

This project demonstrates data visualization techniques using Python and Matplotlib.

## Features
- Load and clean sales data
- Calculate total and average sales
- Identify best-selling product
- Generate charts: Bar, Line, Pie

## Setup Instructions
1. Install Python (3.10 recommended)
2. Install dependencies:
```bash
pip install pandas matplotlib

Run the project:
python main.py
